# 简介

> 使用图像直观展示 `golang` 包依赖关系

# 安装

> `go get github.com/iNevd/graphpkg`

# 依赖

> 依赖[graphviz](http://www.graphviz.org/)

# 使用

> `graphpkg [package1 package2...]`

> `graphpkg net`
